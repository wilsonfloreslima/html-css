Principais formatações.

Negrito, ou destaque.

Nessa frase temos um termo em negrito usando a tag B. (Não semântico)

Nesta frase temos termo de detaque usando a tag STRONG. (Semântica)

Itálico ou enfâse.

Nesta frase temos um termo em itálico, usando a tag i. (Não semântica)

Nesta frase temos um termo com enfâse usando a tag EM. (Sermântica)

!!!!!Lembrar do atalho Ctrl+ Shift+ p!!!!!!.

Texto marcado. (Marca texto)

Podemos criar tambem um texto marcado usando a tag MARK.

Textos grandes e pequenos.

Estamos criando um texto grande e um texto pequeno neste paragrafo.

Texto excluido, ou texto deletado.

Podemos marcar um texto excluido para indicar que ele deve ser ignorado, ou lido mas não considerado.

Texto inserido.

Podemos marcar um texto como inserido para dar uma enfâse e indicar que ele foi adicionado depois.

Podemos tambem sublinhar textos com a tag U, porem essa é uma tag não samântica.

Texto sobrescrito.

Para inserir textos como X5+3, usamos a tag SUP.

Texto subscrito.

Para inserir textos como H2O, usamos a tag SUB.
